# 神木防衛 · 10 關 100 波 戰役設計

**Status**: Design only（先設計、後實作）  
**Genre**: 路線式塔防（Tower Defense）  
**Core**: 守護 **神木**（Delivery Core）  
**Units**: 22 職業卡（可升級 ★1–5）  
**Target**: **10 關 · 共 100 波 · 每關 1 終關 Boss**（部分關卡含中 Boss）

---

## 1. 設計目標

| 目標 | 說明 |
|------|------|
| 教學 → 精通 | 前 3 關教機制，中段考組合，後期考極限佈陣 |
| 每關有「一個問題」 | 例如：雙路、隱形、復活、召喚、換路、護甲 |
| Boss 有讀招 | 不是血厚，要有階段機制 |
| 職業差異有價值 | 破隱／控場／破甲／清線各有關卡 |
| 升級有動力 | 通關楓葉 + 首通，逼玩家養卡 |

## 6. Boss 設計（10 + 可選中 Boss）

### 共通規則

- Boss 漏到神木：**漏傷 5–8**（終章 10）
- Boss 有 **HP 門檻階段**（75% / 50% / 25%）
- Boss 死亡＝該關勝利（當波其餘小怪可清除或一起結束）
- 建議 Boss 血量（基礎，實作再 balance）：

| 關 | Boss | HP | 移速 | 階段機制 |
|----|------|-----|------|----------|
| 01 | 肥肥王 Pig King | 800 | 慢 | 50% 召喚 ribbon_pig×6；一次自癒 30% |
| 02 | 石巨人王 | 1200 | 極慢 | 護甲 50%；破甲後狂暴加速 |
| 03 | 殭屍菇王 | 1000 | 中 | 隱身 5s @50%；死亡分裂 zombie×4 |
| 04 | 蝙蝠王 | 1100 | 中快 | 每 25% 空降 bat×6 |
| 05 | 混混石像鬼 | 1500 | 慢 | 甲 40%→狂暴甲 10% 加速 |
| 06 | 雪吉拉王 | 1600 | 中 | 光環加速小怪；25% 冰封隨機 2 塔 3s（攻速 0） |
| 07 | 刺章魚王 | 1800 | 慢 | 3 觸手召喚；毀觸手停召喚 |
| 08 | 機械豬王 | 2000 | 中 | 鎖定薄弱路線衝鋒 |
| 09 | 幽靈船長 | 2200 | 中 | 階段換路；半血分身（幻影 40% 血） |
| 10 | **殘暴炎魔** | 4000 | 慢 | 見下 |

### Stage 10 · 殘暴炎魔（最終 Boss）三階段

```
Phase 1 (100%–70%)  手臂階段
  - 本體幾乎不動，每 8s 從雙路刷 fire_boar 波
  - 需「有效輸出」削血（有護盾：每 10 擊破一次盾）

Phase 2 (70%–35%)  狂暴
  - 護甲 25%，移速提升
  - 每 20% 召喚 jr_wraith×4 雙路
  - 半血自癒取消（改為召喚 papulatus 影像 15s）

Phase 3 (35%–0%)   終焉
  - 雙路同時刷混合潮
  - 炎魔每 12s 對「隨機一條路」釋放火牆：該路塔攻速 -40% 5s
  - 擊殺即通關戰役
```

**通關獎勵**：🍁 大量 + 稱號「神木守護者」+ 全卡升級折扣（可選）

## 7. 波次編排原則（給實作者）

### 7.1 資料格式（沿用現有）

```js
{
  name: "WAVE_CODE",
  intel: "玩家看的提示",
  groups: [
    { at: 0, path: "workflow", units: [["snail", 8]], interval: 0.85 },
    { at: 3.0, path: "event", units: [["bat", 4]], interval: 0.6 },
  ]
}
```

### 7.2 數量粗標（Stage N Wave W）

```
count ≈ baseCount(W) * stageDensity[N]
baseCount: W1=6 … W9=18 線性
Boss 波: 1 boss + escort 6–12
```

### 7.3 獎勵楓葉（建議）

| 事件 | 楓葉 |
|------|------|
| 清一波 | 6 + W + stageIndex×1 |
| 通關 | 40 + stageIndex×25 |
| 首通 | +80 |
| 三星（神木無傷） | +50（可選） |
